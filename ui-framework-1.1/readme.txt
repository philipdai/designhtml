Allied Agent Center UI Framework
Version 1.1
===================================================

The UI Framework is intended to be a self contained framework. The “ui-framework-x.x” folder can be added anywhere within the application.  Only the following files need to be referenced by the page:

CSS
1. css/bootstrap-x.x/bootstrap.css
2. css/themes/ui-framework-theme.css

JS
1. js/jquery.js
2. js/bootstrap.js
3. Include any other plugins and/or scripts depending on need of application 

For IE 8 Support the following scripts need to be called within a conditional comment in the head of the page
<!-- HTML5 shim and Respond.js IE8 support of HTML5 elements and media queries -->
<!--[if lt IE 9]>
	<script src="ui-framework-x.x/js/html5shiv.js"></script>
	<script src="ui-framework-x.x/js/respond.js"></script>
<![endif]-->
